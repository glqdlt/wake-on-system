배치파일을 윈도우 작업표시줄에 고정하게 하는 팁.

1. 작업 표시줄에 고정시키고자 하는 배치 파일을 만들었다면, 윈도우 탐색기에서 해당 배치파일 위에서 마우스 오른쪽 버튼 클릭 후, 메뉴에서 바로 가기 만들기를 선택해서 바로가기를 만든다.

2. 만들어진 바로가기 위에서 마우스 오른쪽 버튼을 눌러서 속성 창을 띄운다.

3. 속성 창의 바로가기 탭에서, 대상 항목의 배치파일 경로의 앞뒤를 큰따옴표(“)로 감싸고, 앞에 “cmd.exe /C”를 붙인다. 아래와 같은 형태가 된다.

4. 위와 같이 작성하고 확인 버튼을 누르면 해당 바로가기의 아이콘이 톱니바퀴의 모양이서 명령 프롬프트의 아이콘으로 바뀔 것이다. 이 아이콘을 작업 표시줄로 드래그하면 배치파일을 작업 표시줄에 고정시킬 수 있다.

note) 참고로 cmd.exe 파일의 /C 옵션은 “지정된 명령어 수행 후 명령 프롬프트를 종료한다” 는 의미이다. 반대로 계속 유지하는 옵션은 /K
